Инструкции

Выберите вашу платформу для настройки.

📱 Android (Hiddify) ▼
1. Скачайте приложение Hiddify из Google Play.
2. Скопируйте ключ из вкладки "Профиль".
3. Откройте Hiddify, нажмите "+" и выберите "Добавить из буфера обмена".
4. Нажмите огромную кнопку для старта.
🍎 iPhone / iPad (V2Ray Tun) ▼
1. Скачайте V2Ray Tun или Streisand из App Store.
2. Скопируйте ключ.
3. Откройте приложение, нажмите "+" и "Import from Clipboard".
4. Запустите VPN.
💻 Windows (Hiddify) ▼
1. Скачайте Hiddify-Next-Windows с GitHub.
2. Скопируйте ключ из профиля.
3. Вставьте его в программу и нажмите Connect.
🍏 Mac OS (V2RayX) ▼
1. Установите V2RayX или Hiddify для Mac.
2. Скопируйте ссылку подписки.
3. Импортируйте и подключитесь.
